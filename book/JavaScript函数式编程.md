# 函数式编程 - JavaScript

[什么是函数式编程思维](https://www.zhihu.com/question/28292740)

1. 所谓函数，并不是指function，而是数学意义上的函数，即变量的映射，一个函数的值取决于函数参数的值，不依赖其他状态。
2. 在函数中，变量是不可变的，不能多次赋值，没有循环，也就需要用递归(栈)来解决迭代问题。
3. 由于没有可变状态，函数就是引用透明和没有副作用的。不共享状态，也就没有资源争用和锁保护。
